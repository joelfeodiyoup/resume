This is still slightly a work in progress, but essentially I wanted to build a back end platform for hosting browser based games. It's a bit more of an excuse to bring together a few different technologies and refresh my skill with each.

The focus is on the back end but there is a basic/minimal FE to interact with things.

Includes are:

- authentication via session tokens
- authentication on endpoints
- openapi for reference
- APM for performance monitoring
- postgres via prisma
- hexagonal architecture
